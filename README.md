# 💉 Vaccine Prediction Project

## 📌 Objective

Predict whether a person will take H1N1 vaccine using machine learning.

## 📊 Dataset

* Features: Demographics, behavior, health data
* Target: h1n1_vaccine (0 = No, 1 = Yes)

## ⚙️ Tools Used

* Python (Pandas, NumPy)
* Scikit-learn
* Power BI

## 🤖 Model

* Random Forest Classifier

## 📈 Evaluation

* Accuracy
* Classification Report

## 📊 Dashboard

* Prediction Distribution
* Actual vs Predicted
* Confusion Matrix
* Probability Analysis

## 🚀 Conclusion

The model predicts vaccination behavior using survey data and helps in public health insights.
